Creando un repositorio en Github

  1. Crear el repositorio en Github. www.github.com
  2. Crear la carpeta donde se almacenará nuestro proyecto
  3. Abrir la CARPETA (no un archivo) con visual studio code
  4. Abrir la terminal integrada
  5. git init este comando inicializa el respositorio
  6. git remote add origin https://github.com/nombredeusuario/repositorio.git esta instrucción agrega el origen
  7. git add nombredearchivo esta instrucción agrega nuestros cambios a stage, o nuestra piscina de cambios
  8. git commit -m "nombre del commit" agrega un nombre a nuestra piscina de cambios
  9. git push -u origin master crea la rama de master en el repositorio y sube los cambios
  10. Finalmente, cuando seguimos haciendo cambios, ejecutamos: git push (luego de haber agregado los cambios y realizado commit)
